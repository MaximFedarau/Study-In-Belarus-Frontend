# Button

Button - is a basic element of every UI, which can be used as a basic way to interact with interface.

## Usage

```tsx
//Types
import { ReactElement } from 'react';

// Components
import { Button } from 'components/Defaults/Button/Button.component.tsx';

export default function BasicComponent(): ReactElement {
  function onButtonClickHandler() {
    //...some actions
  }
  return (
    <div>
      {/*...some content*/}
      <Button onClick={onButtonClickHandler}>Submit</Button>
    </div>
  );
}
```
